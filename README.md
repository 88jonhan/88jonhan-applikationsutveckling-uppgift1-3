## 88jonhan-applikationsutveckling-uppgift1-3

### Hur du kör programmet

```bash
npm install

npm start
```

- Öppna i expo go via mobil eller simulator

### Komponenter

#### React Native

- View
- Text
- Pressable
- StyleSheet
- TextInput

#### Expo

- Slider
- Checkbox
- Image
- LinearGradient

- [x] Projektmappen innehåller en README.md fil - (läs ovan för mer info)
